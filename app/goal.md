Категории ресурсов содержат только название и список дочерних категорий. Категории
ресурсов могут содержать свойства:
• название
• родительская категория (исключительно указана для элементарных категорий)
• значения ресурса категории по умолчанию.
У каждой категории ресурсов есть значения по умолчанию для упрощения введения
ресурсов в базу. Ресурсы должны обладать следующими свойствами:
• категория,
• название,
• дата покупки,
• срок годности,
• количество,
• единицы измерения (шт, кг, л и т.п.),
• минимальное количество.
Минимальное количество - это значение, при достижении которого ресурс считается на
исходе и нужно его пополнить.
Состояния могут быть следующими:
• просроченный (срок годности истёк)
• доступный к использованию (срок годности больше недели)
• срок годности скоро истечёт (срок годности меньше недели)
