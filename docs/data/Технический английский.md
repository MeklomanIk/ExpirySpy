# Технический английский

## Пример заполнения термина

| EN         | RU                      | Значение                          |
| ---------- | ----------------------- | --------------------------------- |
| term in EN | термин на русском языке | определение или пояснение термина |

## Сбои и ошибки

| EN        | RU             | Значение                                      |
| --------- | -------------- | --------------------------------------------- |
| typo      | опечатка       |                                               |
| criteria  | критерий       |                                               |
| error     | ошибка         | не соответствие кода программы её цели, идее  |
| warning   | предупреждение | не соответствие программы анализатору IDE     |
| exception | исключение     | непредвиденная ситуация при запуске программы |

## Требования, соглашения, критерии

| EN          | RU         | Значение                                   |
| ----------- | ---------- | ------------------------------------------ |
| requirement | требование | внешнее условие, которое нужно выполнить   |
| convention  | соглашение | принятый стандарт, стиль выполнения задачи | 

## Теория языков программирования

| EN        | RU         | Значение                                                                    |
| --------- | ---------- | --------------------------------------------------------------------------- |
| syntax    | синтаксис  | правила сочетания символов                                                  |
| paradigm  | парадигма  | совокупность методов и способов программировать, не зависящих от синтаксиса |
| semantic  | семантика  | правила сочетания слов                                                      |
| pragmatic | прагматика | правила сочетания конструкций                                               |
|           |            |                                                                             |

## Система контроля версий Git

| EN    | RU              | Значение                           |
| ----- | --------------- | ---------------------------------- |
| major | главный         | кардинальное изменение программы   |
| minor | вспомогательный | небольшое изменение программы      |
| patch | заплатка        | незначительное изменения программы |
| fix   | исправление     |                                    |
