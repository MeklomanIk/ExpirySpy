# Visual Studio Code

## Горячие клавиши

| Команда       | Описание                                                             |
| -------------------- | ---------------------------------------------------------------------------- |
| `Ctrl + Alt + F`   | Форматировать документ                                  |
| `Ctrl + P`         | Быстрый переход к файлу                                  |
| `Ctrl + Shift + O` | Быстрый переход к символу в файле                |
| `Ctrl + Shift + M` | Открыть панель проблем                                   |
| `ПКМ`           | Открыть контекстное меню                               |
| `Ctrl + /`         | Комментировать/раскомментировать строку |
|                      | Окрыть настройки                                              |
