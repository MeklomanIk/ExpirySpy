# Quarto

## Для чего используется

Quarto — это современная система создания документов, которая позволяет создавать отчёты, презентации, книги и веб-сайты с использованием простого синтаксиса на основе Markdown. Quarto поддерживает интеграцию с различными языками программирования, такими как R, Python и Julia, что делает его мощным инструментом для научных исследований, анализа данных и технической документации.

## Основные возможности Quarto

- создание документов
  - с функцией вставки и выполнения кода на различных языках программирования (R, Python, Julia и др.);
  - экспортом в PDF, HTML, Word и другие форматы;
  - с разметкой документа на MD, Jupyter Notebook, Typst и др.
  - с поддержкой графиков и диаграмм (Mermaid, Graphviz, PlantUML, Vega, Plotly, D3.js и др.);
  - с возможностью создания интерактивных элементов (формы, виджеты и др.);
- настройка стилей и шаблонов документов;
- поддержка управления версиями и совместной работы над документами;
